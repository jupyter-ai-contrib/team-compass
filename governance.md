# Governance

This project is currently in a "governance confusing state" :)

> Technically [jupyter-ai](https://github.com/jupyterlab/jupyter-ai/) (the package) is a repo under jupyterlab governance.
>
> But, I think most folks agree that Jupyter AI (the idea) should become it's own subproject.
> And [jupyter-ai](https://github.com/jupyterlab/jupyter-ai/) the repo (and all other repos in the jupyter-ai-contrib org) should be under that subproject's purview.
